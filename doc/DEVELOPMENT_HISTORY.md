## Project History

This document records the development background and evolution of the Sakura Gojuon (Japanese 50 Sounds Learning) project.

### Initial Motivation

The initial idea was to attempt creating a [Reveal.js](https://revealjs.com/) based Japanese 50 sounds learning slide deck within the **Antigravity** environment. The goal was to explore how to quickly build interactive teaching resources through collaboration between plain text files and an AI Agent.

### Development Stages

#### 1. Foundation
- Started with plain text files (Markdown).
- Established the basic structure for Hiragana and Katakana slides.

#### 2. Agent Collaboration & Feature Expansion
- Introduced **Web Speech API** through Agent modifications, implementing click-to-pronounce functionality.
- Integrated **dmak.js** and **KanjiVG** to add stroke order animation, enhancing learning effectiveness.

#### 3. Visual Optimization
To improve user experience and aesthetics, several visual adjustments were made:

- **Navigation Bar**: Added a global navigation bar at the bottom for quick chapter switching.
- **Background Design**: Added a subtle sakura watermark background.
- **Transitions**: Implemented "Sakura Blizzard" particle effects that trigger during slide transitions to create a cherry blossom falling atmosphere.

## 專案發展歷程

本文件記錄了 Sakura Gojuon (日語五十音學習) 專案的發展背景與演進過程。

### 初始動機

最初的構想是為了嘗試在 **Antigravity** 環境中，製作一份基於 [Reveal.js](https://revealjs.com/) 的日語五十音教學投影片。目標是探索如何透過純文字檔案與 AI Agent 協作，快速構建出具有互動性的教學資源。

### 發展階段

#### 1. 基礎架構
- 專案起始於純文字檔案 (Markdown)。
- 建立了基本的 Hiragana 與 Katakana slide 結構。

#### 2. Agent 協作與功能擴充
- 透過 Agent 進行修改，引入了 **Web Speech API**，實現了點擊發音的功能。
- 進一步整合了 **dmak.js** 與 **KanjiVG**，增加了筆順動畫展示功能，提升學習效果。

#### 3. 視覺優化
為了提升使用體驗與美感，進行了多項視覺調整：

- **導覽列 (Navigation Bar)**：新增底部的全域導覽列，方便使用者快速切換章節。
- **背景設計 (Background)**：加入了淡雅的櫻花浮水印背景。
- **轉場動畫 (Transition)**：實作了「櫻吹雪」(Sakura Blizzard) 的粒子特效，在投影片切換時營造出櫻花飄落的氛圍。
